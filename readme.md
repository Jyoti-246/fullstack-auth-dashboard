# FullStack Auth Dashboard - Jyoti Kedia

**Live Demo:** [View on Vercel](https://fullstack-auth-dashboard-lac.vercel.app)

## Overview

A **Full-Stack Authentication Dashboard** built with **React.js (frontend)** and **Node.js/Express (backend)**. Includes user authentication, task management, and a responsive UI.

---

## Features

- User sign-up, login, logout
- Task creation, update, delete
- RESTful API integration
- Responsive UI
- Backend logging in `logs/` folder

---

## Tech Stack

- Frontend: React.js, Vite, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: PostgreSQL

---

## Installation

### Backend

```bash
cd backend
npm install
npm start
```
